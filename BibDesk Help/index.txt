# BibDesk

## About BibDesk

BibDesk 1.0 is a BibTeX editor and reference manager. Because it is a
BibTeX editor, it is important to understand at least a few things
about the BibTeX file format.

* If you already have a BibTeX database and are moving from hand-editing
  or another BibTeX editor, start at the [quick start](quickstart.html) page.

* If you are just getting started keeping track of references or are
  moving from a program such as [EndNote](http://www.endnote.com) or
  another reference manager, see the [quick intro to BibTeX](bibtex.html).
